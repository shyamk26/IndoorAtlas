Inherits from NSObject

Overview

IndoorAtlas class is a singleton representing the API interface. Use the singleton for obtaining SDK version information, authenticating the session, fetching floor plans and instantiating and IndoorAtlasPositioner objects.

fetchFloorplanImage:completion:

Fetch image file data for floor plan object.

+ (id<IndoorAtlasFetch>)fetchFloorplanImage:(IndoorAtlasFloorplan *)floorplan completion:(void ( ^ ) ( NSData *data , NSError *error ))completionBlock

Parameters

floorplan

A floor plan to fetch image file data for.

completionBlock

A block to be called when image download is completed.

  • data: Image file data. If an error occured, this parameter is nil.
  • error: If an error occured, this object describes the error. If the operation completed successfully, this value is nil.

IndoorAtlasFetch instance for the request.
